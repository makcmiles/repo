```
Файл Vagrant находится в репозитории.
Копируются файлы из директории config в ВМ, потом скриптом копируются в свои директории.
Выполняются два скрипта: 
- pre-script.sh выполняет установку необходимых инструментов и скачивание первоночальных пакетов - nginx и openssl.
- script.sh выполняет основную настройку репозитория, готовит пакеты.

P.S. Задача максимум конечно обойтись без скачивания конфигурационных файлов с хостовой машины, т.к. conf и spec формируются в процессе установки и настройки, а значит можно внести туда изменения на определённом этапе скрипта. Но пока недостаточно практики и знаний для такого.
```
